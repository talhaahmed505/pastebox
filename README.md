# pastebox

## pastebox is a web app for copying and pasting text between different machines without a share link. It uses web sockets to broadcast changes to all current clients

Pastebox supports post commands sent to /cli for the same functionality as the web page.

curl {pastebox-url}/cli to read the current display text
curl -X POST {pastebox-url}/cli --data {data} to update display text
